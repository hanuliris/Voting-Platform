# DTO (Data Transfer Objects) Package

**Purpose**: Request/Response objects for API communication

**Files**:
- `LoginRequest.java` - Login credentials
- `RegisterRequest.java` - Registration data
- `AuthResponse.java` - JWT token response
- `PollRequest.java` - Poll creation data
- `VoteRequest.java` - Vote submission data
- `VoteResultDTO.java` - Poll results data
- `UserDTO.java` - User information
